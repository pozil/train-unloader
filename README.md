# Robotics Ridge - Train unloader

## Setup instructions


### App installation
Install with:<br/>
`npm install`

Edit and configure the `.env` file:
```
domain='https://test.salesforce.com'
callbackUrl='http://localhost:5000/'
consumerKey=''
consumerSecret=''
apiVersion='v45.0'

sfdcUsername=''
sfdcPassword=''
```

Build with:<br/>
`npm run build`

Start with:<br/>
`npm start`
